## 🚀 LeetCode Practice & GitHub Contributions

I regularly solve Data Structures and Algorithms (DSA) problems on LeetCode to strengthen my problem-solving skills and improve my understanding of algorithms and data structures.

### What I Focus On

* Arrays & Strings
* Linked Lists
* Stacks & Queues
* Trees & Graphs
* Dynamic Programming
* Greedy Algorithms
* Recursion & Backtracking

### My Approach

For each problem, I:

1. Analyze the problem statement and constraints.
2. Develop a brute-force solution to understand the logic.
3. Optimize the solution for better time and space complexity.
4. Implement the final solution in Java.
5. Document important observations and learning points.

### GitHub Contributions

All solved problems are regularly committed to GitHub with:

* Well-structured Java solutions
* Meaningful commit messages
* Clean and readable code
* Explanations of approaches and complexities
* Continuous progress tracking

This practice helps me build consistency, improve coding interview preparation, and maintain a strong record of my learning journey.
